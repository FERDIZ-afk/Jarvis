## Getting Started

This project require NodeJS v12 or above

### Install
Clone this project



```bash
> git clone https://github.com/Rishabh053/Jarvis
> cd Jarvis
```

Install the dependencies:

```bash
> npm install
```

### Usage
 Run the Jarvis

```bash
> npm start
```
 
After running it you need to scan the QR Code in the Terminal
